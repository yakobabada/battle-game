# battle-game
PHP application that simulates a battle between two combatants. It's PHP application, it's build on symfony 4 framework.

## Requirement:

PHP 7

## Installation

- `git clone https://github.com/yakobabada/battle-game.git`
- `cd battle-game/`
- `composer install`

## Start the battle

- `bin/console app:play-game`
